# Workstream - User Beta Expert Tester

## Scope

- End-to-end user journey quality
- Error recovery and fallback transparency
- Desktop/mobile behavior under realistic usage

## Checklist

- [ ] Test dashboard loading with normal, degraded, and failed API conditions.
- [ ] Test trends list filtering, pagination, and detail page transitions.
- [ ] Test early-signals and opportunities pages for continuity and confidence signals.
- [ ] Validate search responsiveness and failure messaging.
- [ ] Validate mobile navigation and task completion on small viewport.

## Required Evidence

- Scenario matrix with pass/fail by browser + viewport
- Reproduction steps for high-impact UX defects
- Prioritized user-impact defect list
