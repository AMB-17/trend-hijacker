# Workstream - UI/UX Expert

## Scope

- Information architecture and readability
- Interaction consistency and state feedback
- Accessibility and keyboard usability

## Checklist

- [ ] Review hierarchy clarity on dashboard, trends, opportunities, and early-signals pages.
- [ ] Validate loading, error, and empty states for clarity and confidence.
- [ ] Validate consistency in cards, filters, and status badges.
- [ ] Audit ARIA labels, focus visibility, keyboard traversal, and contrast.
- [ ] Validate mobile layout coherence and navigation affordances.

## Required Evidence

- Heuristic findings with severity and recommended fixes
- Accessibility checklist with WCAG-oriented notes
- Before/after UI recommendations for top 5 issues
